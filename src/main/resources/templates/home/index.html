O que é o Muralize?

Este é um espaço feito para tirar as ideias do papel. O Muralize nasceu como um projeto de estudo para colocar a mão na massa com o ecossistema Java e ver como as coisas realmente funcionam.

A ideia é simples: um lugar onde a gente possa trocar mensagens, deixar avisos ou só testar se o banco de dados está aguentando o tranco. Tudo de forma colaborativa e, claro, bem organizada.

Tudo no lugar
Arquitetura limpa e organizada em camadas.

Dados Vivos
Persistência real com Spring Data JPA.

Seguro (na medida)
Acesso controlado por cookies e interceptors.

Páginas Ágeis
Renderização dinâmica com Thymeleaf.
